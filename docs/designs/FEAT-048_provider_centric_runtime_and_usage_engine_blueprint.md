## 3. Complete Class & Module Design

### `connectors/base.py` — ProviderConnector (ABC)

- **Responsibilities**: Define the contract all provider connectors must implement
- **Constructor Parameters**: `config: dict = {}` (overrides for paths and settings)
- **Public Methods**:
  - `def detect(self) -> DetectedProvider | None` — check if this provider is installed on current system
  - `def discover_conversations(self) -> list[str]` — return list of conversation IDs available
  - `def parse_conversation(self, conv_id: str) -> list[NormalizedUsageRecord]` — parse and normalize usage for given conversation
  - `def get_diagnostics(self) -> DiagnosticsResult` — return status, paths, errors for diagnostics panel
  - `def get_provider_name(self) -> str` — return canonical provider name (e.g., "antigravity")
- **Internal Methods**:
  - `def _get_default_paths(self) -> list[str]` — OS-aware default log/brain root paths
  - `def _resolve_path(self, env_var: str, default: str) -> str` — env override + default
- **Dependencies**: `abc`, `dataclasses`, `normalizer.NormalizedUsageRecord`
- **Extension Points**: Subclasses override `_get_default_paths()` and all `@abstractmethod` methods

### `connectors/__init__.py` — ConnectorRegistry

- **Responsibilities**: Registry and router for all provider connectors
- **Constructor Parameters**: None (singleton-like, initialized with `build_default_registry()`)
- **Public Methods**:
  - `def register(self, connector: ProviderConnector) -> None`
  - `def detect_all(self) -> list[DetectedProvider]` — run detect() on all registered connectors
  - `def get_connector(self, provider_name: str) -> ProviderConnector | None`
  - `def parse(self, provider_name: str, conv_id: str) -> list[NormalizedUsageRecord]`
  - `def diagnose_all(self) -> list[DiagnosticsResult]`
- **Internal Methods**:
  - `def _load_manifest(self) -> list[str]` — load `connectors.json` manifest
- **Dependencies**: `connectors/base.py`, all connector subclasses
- **Extension Points**: Call `register()` with new connector instance

### `connectors/antigravity.py` — AntigravityConnector

- **Responsibilities**: Read Antigravity IDE transcripts from `~/.gemini/antigravity-ide/brain/`
- **Constructor Parameters**: `config: dict = {}` (override `brain_root` path)
- **Public Methods**: All from `ProviderConnector` ABC
- **Internal Methods**:
  - `def _find_transcript_file(self, conv_id: str) -> str | None`
  - `def _parse_jsonl(self, file_path: str, start_hash: str) -> tuple[list, str]` — returns (records, new_hash)
- **Dependencies**: `transcript_engine.IncrementalTranscriptReader`, `normalizer.UsageNormalizer`
- **Default Path**: `os.path.expanduser("~/.gemini/antigravity-ide/brain")`; env override: `ANTIGRAVITY_BRAIN_ROOT`

### `connectors/claude_code.py` — ClaudeCodeConnector

- **Responsibilities**: Read Claude Code usage logs from OS-specific config directories
- **Default Paths** (OS-aware):
  - Windows: `%APPDATA%\Claude\logs\`
  - macOS: `~/Library/Application Support/Claude/logs/`
  - Linux: `~/.config/claude/logs/`
- **Internal Methods**: `_parse_claude_log_entry(entry: dict) -> NormalizedUsageRecord | None`
- **Accuracy Source**: `transcript_parsed` (Claude does not expose token counts in JSONL format currently)

### `connectors/cursor.py` — CursorConnector

- **Responsibilities**: Read Cursor AI log files
- **Default Paths** (OS-aware):
  - Windows: `%APPDATA%\Cursor\logs\`
  - macOS: `~/Library/Application Support/Cursor/logs/`
  - Linux: `~/.config/Cursor/logs/`
- **Accuracy Source**: `transcript_parsed`

### `connectors/vscode_agents.py` — VSCodeAgentsConnector

- **Responsibilities**: Read VS Code AI agent extension logs
- **Default Paths**: `{vscode_user_data}/logs/`; resolved via `code --list-extensions` or env override `VSCODE_LOGS_PATH`
- **Accuracy Source**: `transcript_parsed`

### `transcript_engine.py` — IncrementalTranscriptReader

- **Responsibilities**: Read JSONL transcript files incrementally using file hash tracking
- **Constructor Parameters**: `db_conn: sqlite3.Connection` (for cache reads/writes)
- **Public Methods**:
  - `def read_new_lines(self, file_path: str) -> list[dict]` — returns only new lines since last read
  - `def get_file_hash(self, file_path: str) -> str` — MD5 of last-read byte position + size
  - `def reset(self, file_path: str) -> None` — force full re-read on next call
- **Internal Methods**:
  - `def _load_cursor(self, file_path: str) -> int` — load last byte position from `transcript_cursors` SQLite table
  - `def _save_cursor(self, file_path: str, position: int, file_hash: str) -> None`
- **Dependencies**: `hashlib`, `sqlite3`, `db.py` for `transcript_cursors` table
- **New SQLite Table Required**: `transcript_cursors(file_path TEXT PRIMARY KEY, byte_position INTEGER, file_hash TEXT, updated_at TEXT)`

### `normalizer.py` — UsageNormalizer

- **Responsibilities**: Map provider-specific dicts to `NormalizedUsageRecord` dataclass
- **Public Methods**:
  - `def normalize(self, raw: dict, provider: str, accuracy_source: str) -> NormalizedUsageRecord`
  - `def validate(self, record: NormalizedUsageRecord) -> list[str]` — return list of validation errors
- **NormalizedUsageRecord dataclass fields**:
  ```python
  provider: str                  # "antigravity" | "claude_code" | "cursor" | "vscode_agents"
  model: str                     # e.g. "gemini-2.5-flash"
  conversation_id: str
  request_id: str                # unique per request (conv_id + step_index)
  timestamp: str                 # ISO8601
  input_tokens: int
  output_tokens: int
  cache_read_tokens: int
  cache_write_tokens: int
  thinking_tokens: int
  total_tokens: int              # input + output
  duration_ms: float
  estimated_cost_usd: float
  accuracy_source: str           # "provider_reported" | "transcript_parsed" | "derived" | "estimated" | "unknown"
  raw_payload: dict              # original provider dict preserved
  ```

### `cost_engine.py` — CostEngine

- **Responsibilities**: Load `pricing.json`, calculate costs per request
- **Constructor Parameters**: `pricing_path: str` (defaults to `skills/workflow-runtime/data/pricing.json`)
- **Public Methods**:
  - `def calculate(self, provider: str, model: str, input_tokens: int, output_tokens: int, cache_read_tokens: int = 0, cache_write_tokens: int = 0) -> CostResult`
  - `def get_pricing(self, provider: str, model: str) -> ModelPricing | None`
  - `def is_stale(self, days: int = 30) -> bool` — check if pricing.json is older than N days
  - `def get_version(self) -> str`
- **CostResult dataclass**: `{ cost_usd: float, breakdown: { input_cost, output_cost, cache_read_cost, cache_write_cost } }`
- **Dependencies**: `json`, `datetime`, `data/pricing.json`
- **Error handling**: If provider/model not found → returns `estimated` with generic per-token rate

### `provider_engine.py` — Standalone CLI

- **Responsibilities**: Single standalone entry point for all provider operations
- **Subcommands** (see Section 10 for full CLI contracts):
  - `detect` → JSON list of DetectedProvider
  - `parse --provider X --conv-id Y` → JSON list of NormalizedUsageRecord
  - `pricing --provider X --model Y --input N --output M [--cache-read R] [--cache-write W]` → JSON CostResult
  - `diagnose` → JSON DiagnosticsReport
- **Dependencies**: All connectors, normalizer, cost_engine, db
- **MUST work standalone** — no import of `workflow_runtime.py` or AIWF-specific modules

### `event_bus.py` — RuntimeEventBus

- **Responsibilities**: Capture, store, and replay runtime events
- **Public Methods**:
  - `def emit(self, event: RuntimeEvent) -> None` — saves to `runtime_events` SQLite table
  - `def subscribe(self, event_type: str, handler: Callable) -> None` — in-process handlers
  - `def replay(self, conversation_id: str, since: str = None) -> list[RuntimeEvent]`
- **RuntimeEvent dataclass**: `{ id: str, timestamp: str, conversation_id: str, provider: str, event_type: str, event_data: dict }`
- **Event types**: `skill_start`, `skill_complete`, `skill_fail`, `checkpoint_advance`, `tool_call`, `build`, `test`, `shell`

### `log_collectors.py` — IDE Log Collectors

- **Responsibilities**: Tail IDE-specific log files and emit RuntimeEvents via EventBus
- **Class**: `AntigravityLogCollector`, `ClaudeCodeLogCollector` — both extend `BaseLogCollector`
- **Public Methods**: `def start_tailing(self, conv_id: str) -> None`, `def stop(self) -> None`

### `WorkflowMetadataAdapter` (inside `context.py`)

- **Responsibilities**: Optional AIWF integration — reads `.agents/state/` files if present
- **Public Functions** (module-level, not class):
  - `def get_workflow_metadata(state_dir: str = ".agents/state") -> dict | None` — returns `{checkpoint, current_skill, work_item, status, memory, rag}` or `None` if state_dir absent

## 4. Detailed Interface Contracts

### `ProviderConnector.detect() -> DetectedProvider | None`

- **Parameters**: None
- **Return Types**: `DetectedProvider(provider_name: str, path: str, version: str, status: str)` or `None`
- **Exceptions**: Never raises — returns `None` on any detection failure
- **Validation Rules**: `path` must exist and be readable before returning non-None
- **Compatibility Notes**: Must work on Windows (backslash paths), macOS, Linux

### `ConnectorRegistry.parse(provider_name, conv_id) -> list[NormalizedUsageRecord]`

- **Parameters**: `provider_name: str` (exact match against registered names), `conv_id: str`
- **Return Types**: `list[NormalizedUsageRecord]` — empty list if no data found
- **Exceptions**: `ConnectorNotFoundError(provider_name)` if provider not registered; `TranscriptNotFoundError(conv_id)` if transcript missing
- **Validation Rules**: `conv_id` must be non-empty string

### `CostEngine.calculate(...) -> CostResult`

- **Parameters**: `provider: str`, `model: str`, `input_tokens: int`, `output_tokens: int`, `cache_read_tokens: int = 0`, `cache_write_tokens: int = 0`
- **Return Types**: `CostResult(cost_usd: float, breakdown: dict, model_matched: bool, fallback_used: bool)`
- **Exceptions**: Never raises — uses generic fallback rate if model not found
- **Validation Rules**: All token counts must be `>= 0`

### `IncrementalTranscriptReader.read_new_lines(file_path) -> list[dict]`

- **Parameters**: `file_path: str` — absolute path to `.jsonl` file
- **Return Types**: List of parsed JSON line dicts (only new lines since last call)
- **Exceptions**: `FileNotFoundError` propagated; `PermissionError` caught + logged to diagnostics
- **Validation Rules**: file must be UTF-8 readable; malformed JSON lines skipped silently

## 6. Database & Storage Design

### New Tables (Additive — `db.py init_db_schema()`)

#### `transcript_cursors`
```sql
CREATE TABLE IF NOT EXISTS transcript_cursors (
    file_path   TEXT PRIMARY KEY,
    byte_pos    INTEGER NOT NULL DEFAULT 0,
    file_hash   TEXT NOT NULL DEFAULT '',
    updated_at  TEXT NOT NULL
);
```

#### `runtime_events`
```sql
CREATE TABLE IF NOT EXISTS runtime_events (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    event_id        TEXT NOT NULL UNIQUE,
    timestamp       TEXT NOT NULL,
    conversation_id TEXT NOT NULL,
    provider        TEXT NOT NULL,
    event_type      TEXT NOT NULL,
    event_data_json TEXT NOT NULL
);
CREATE INDEX IF NOT EXISTS idx_runtime_events_conv ON runtime_events (conversation_id);
CREATE INDEX IF NOT EXISTS idx_runtime_events_type ON runtime_events (event_type);
```

#### `connector_diagnostics`
```sql
CREATE TABLE IF NOT EXISTS connector_diagnostics (
    id                  INTEGER PRIMARY KEY AUTOINCREMENT,
    timestamp           TEXT NOT NULL,
    provider            TEXT NOT NULL,
    status              TEXT NOT NULL,
    detected_path       TEXT,
    error_message       TEXT,
    accuracy_confidence TEXT NOT NULL
);
CREATE INDEX IF NOT EXISTS idx_connector_diag_provider ON connector_diagnostics (provider);
```

### Modified Columns (Additive — `ALTER TABLE ADD COLUMN IF NOT EXISTS` pattern)

```sql
-- provider_requests table: two new columns
ALTER TABLE provider_requests ADD COLUMN accuracy_source TEXT DEFAULT 'estimated';
ALTER TABLE provider_requests ADD COLUMN raw_payload TEXT;
```

**Migration execution**: Inside `init_db_schema()`, wrap in `try/except sqlite3.OperationalError` matching existing pattern at `db.py:47–51`.

### Indexes
- `idx_runtime_events_conv` on `runtime_events(conversation_id)` — O(log n) lookup by conversation
- `idx_runtime_events_type` on `runtime_events(event_type)` — filter by event type
- `idx_connector_diag_provider` on `connector_diagnostics(provider)` — per-provider diagnostics lookup

### Relationships / Constraints
- `runtime_events.conversation_id` soft-references `provider_requests.conversation_id` (no FK — SQLite FK enforcement not required)
- `transcript_cursors.file_path` is primary key — unique per file

### Migration & Rollback Strategy
- **All migrations additive** — no `DROP TABLE`, no `ALTER TABLE DROP COLUMN`
- **Pre-migration backup**: `shutil.copy(db_path, db_path + ".bak")` before any schema changes
- **Rollback SQL**: Restore `.bak` file; new tables/columns have no data dependencies on old code

## 7. Cache Architecture

- **Cache Keys**: `transcript_cursors.file_path` (absolute path as primary key)
- **Invalidation Rules**: Cache invalidated when `file_hash` changes (file modified) OR when `reset()` explicitly called
- **TTL**: No time-based TTL — invalidation is hash-driven only
- **Hash Strategy**: MD5 of `(file_path, byte_position, file_mtime_ns)` — fast, collision-resistant for file tracking
- **Provider versioning & stale detection**: `pricing.json` version checked at `CostEngine` init; `is_stale(days=30)` returns True if `updated_at` > 30 days ago → DiagnosticsPanel shows amber warning
- **Warmup**: First call always reads full file; subsequent calls seek to `byte_pos`
- **Cleanup**: `transcript_cursors` rows older than 90 days pruned on init (to handle deleted conversations)

## 8. Error Model

### `ConnectorNotFoundError`
- **Trigger**: `ConnectorRegistry.parse()` called with unregistered `provider_name`
- **Recovery**: Log to `connector_diagnostics`, return empty list
- **Retry Policy**: No retry — misconfiguration error
- **Logging**: `WARNING` level, keys: `provider_name`, `registered_providers`

### `TranscriptNotFoundError`
- **Trigger**: `IncrementalTranscriptReader.read_new_lines()` called with non-existent path
- **Recovery**: Log path to `connector_diagnostics` with `status="NOT_FOUND"`, return empty list
- **Retry Policy**: No retry — caller should check before calling
- **Logging**: `INFO` level (expected case when conversation not started yet)

### `PricingModelNotFoundError`
- **Trigger**: `CostEngine.calculate()` called with unknown provider/model combination
- **Recovery**: Use `fallback` block from `pricing.json`; set `fallback_used=True` in CostResult
- **Logging**: `DEBUG` level, keys: `provider`, `model`

### `TranscriptParseError`
- **Trigger**: Malformed JSON line in `.jsonl` file
- **Recovery**: Skip line, continue parsing, log to diagnostics
- **Retry Policy**: No retry — corrupted lines skipped permanently
- **Logging**: `WARNING` level, keys: `file_path`, `line_number`, `raw_content[:100]`

### `FileLockedError` (Windows)
- **Trigger**: `PermissionError` when opening transcript file locked by IDE process
- **Recovery**: Exponential backoff: retry 3 times at 100ms, 200ms, 400ms; then log to diagnostics and return empty list
- **Logging**: `WARNING` level, keys: `file_path`, `attempt`

- **Syntax**: `python provider_engine.py diagnose [--format json]`

- **Syntax**: `python workflow_runtime.py provider <subcommand> [args...]`

- **Exit Codes**: Passes through provider_engine.py exit code

## 11. Sequence Flows

### Normal Execution Flow (Extension Dashboard Refresh)

1. Extension `updateSessionData()` triggered by file watcher
2. Extension calls `spawn("provider_engine.py parse --provider antigravity --conv-id {conv_id}")`
3. `provider_engine.py` → `ConnectorRegistry.parse("antigravity", conv_id)`
4. `AntigravityConnector.parse_conversation(conv_id)`
5. `IncrementalTranscriptReader.read_new_lines(transcript_path)` → checks `transcript_cursors` SQLite table
   - **Cache Hit**: Returns only new lines since last byte_pos
   - **Cache Miss** (first run): Reads full file, saves byte_pos and hash
6. `TranscriptParser` parses each line → raw dicts
7. `UsageNormalizer.normalize()` → `NormalizedUsageRecord[]`
8. `CostEngine.calculate()` for each record
9. Records written to SQLite `provider_requests` with `accuracy_source="transcript_parsed"`
10. JSON output to stdout
11. Extension receives JSON, updates all metric cards with AccuracyBadge

### AIWF Optional Merge Flow

1. Extension calls `workflow_runtime.py usage` (existing) → returns `.agents/state/` based data
2. Extension merges workflow metadata (checkpoint, skill, work_item) from AIWF data
3. Extension uses provider_engine.py data for token/cost stats (wins over AIWF estimates)
4. If `.agents/` absent: workflow cards show "N/A — AIWF not installed"; usage cards show provider data

### DiagnosticsPanel Refresh Flow

1. Extension calls `spawn("provider_engine.py diagnose")`
2. `ConnectorRegistry.diagnose_all()` → per-connector `get_diagnostics()`
3. Each connector checks path existence, file permissions, last parsed timestamp
4. Results written to `connector_diagnostics` SQLite table
5. JSON DiagnosticsReport returned to Extension
6. DiagnosticsPanel updated: green (found), amber (stale pricing), red (not found / error)

### Cache Miss → Full Parse Flow (First Run, 50MB transcript)

1. `IncrementalTranscriptReader.read_new_lines(file_path)`
2. No entry in `transcript_cursors` → `byte_pos = 0`
3. Open file, read all lines in streaming fashion (1MB buffer)
4. Parse each JSONL line → skip malformed lines (log warning)
5. Save final `byte_pos` and `file_hash` to `transcript_cursors`
6. Return all parsed records
7. Target: < 5 seconds for 50MB file

## 12. Security & Safety

- **Workspace Boundary**: `provider_engine.py` reads transcript files in read-only mode (`open(f, 'r')`). Never writes to transcript files.
- **Path Validation**: All paths resolved via `os.path.realpath()` before access. Paths must start with allowed prefixes (BRAIN_ROOT, OS config dirs). Reject any path containing `..` traversal.
- **Write Restrictions**: `provider_engine.py` writes only to `project_runtime.db` and `global_runtime.db`. Never modifies files in workspace source tree.
- **Webview Exposure**: DiagnosticsPanel shows only: provider name, status, relative path prefix, error type. Never shows absolute filesystem paths (masked as `~/.gemini/...`) and never shows message content.
- **Rollback safety**: All SQLite writes wrapped in transactions. On `except` → `conn.rollback()` before close.

## 15. File-Level Implementation Contracts

### `skills/workflow-runtime/scripts/connectors/base.py`
- **Purpose**: Abstract base class establishing the ProviderConnector contract
- **Owner**: Architect
- **Inputs**: `config: dict` — optional path overrides
- **Outputs**: `DetectedProvider | None`, `list[NormalizedUsageRecord]`, `DiagnosticsResult`
- **Dependencies**: `abc`, `dataclasses`
- **Implementation Notes**: Use `@abstractmethod` decorator. `_get_default_paths()` must handle Windows (backslash), macOS, Linux. Never raise in `detect()` — catch all exceptions and return `None`.

### `skills/workflow-runtime/scripts/connectors/__init__.py`
- **Purpose**: ConnectorRegistry — loads manifest, registers all enabled connectors, routes calls
- **Owner**: Architect
- **Inputs**: `connectors.json` manifest path (default: same directory as `__init__.py`)
- **Outputs**: `list[DetectedProvider]`, `list[NormalizedUsageRecord]`, `list[DiagnosticsResult]`
- **Dependencies**: `base.py`, `importlib`, `json`
- **Implementation Notes**: `build_default_registry()` factory function creates pre-populated registry. `importlib.import_module()` + `getattr()` for dynamic class loading from manifest.

### `skills/workflow-runtime/scripts/transcript_engine.py`
- **Purpose**: Incremental JSONL reader with SQLite cursor tracking
- **Owner**: Coder
- **Inputs**: `file_path: str`, SQLite `transcript_cursors` table
- **Outputs**: `list[dict]` — parsed JSON lines (new content only)
- **Dependencies**: `json`, `hashlib`, `sqlite3`, `db.py` (for cursor persistence)
- **Implementation Notes**: Use `file.seek(byte_pos)` to resume from last position. Use `file.tell()` after each line to track position. Handle `UnicodeDecodeError` with `errors='replace'`. Never seek backwards except on `reset()`.
- **Risks**: Windows file locking — wrap open in retry with exponential backoff

### `skills/workflow-runtime/scripts/normalizer.py`
- **Purpose**: Map raw provider dicts to NormalizedUsageRecord dataclass
- **Owner**: Coder
- **Inputs**: `raw: dict`, `provider: str`, `accuracy_source: str`
- **Outputs**: `NormalizedUsageRecord` dataclass instance
- **Dependencies**: `dataclasses`
- **Implementation Notes**: Use `dataclasses.dataclass` with `__post_init__` for validation. `total_tokens = input_tokens + output_tokens` always computed, never trusted from raw. `accuracy_source` immutable after construction.

### `skills/workflow-runtime/scripts/cost_engine.py`
- **Purpose**: Load pricing.json and calculate per-request costs
- **Owner**: Coder
- **Inputs**: `pricing.json` path, provider/model/token counts
- **Outputs**: `CostResult` dataclass
- **Dependencies**: `json`, `datetime`, `pathlib`
- **Implementation Notes**: Cache parsed `pricing.json` in memory after first load. Check `updated_at` on every `is_stale()` call without reloading full file. `fallback_used` flag must be set when fallback pricing applied.

### `skills/workflow-runtime/scripts/provider_engine.py`
- **Purpose**: Standalone CLI — single entry point for all provider operations
- **Owner**: Coder
- **Inputs**: CLI arguments via `argparse`
- **Outputs**: JSON to stdout, errors to stderr
- **Dependencies**: `connectors/`, `normalizer.py`, `cost_engine.py`, `db.py`
- **Implementation Notes**: `if __name__ == "__main__":` guard. Use `sys.exit(code)` for exit codes. Print only valid JSON to stdout — no debug prints. `--format json` is the default and only supported format for now.
