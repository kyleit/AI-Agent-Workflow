# Information Architecture — AIWF Runtime Platform v1

Cấu trúc cây thư mục tài liệu chính thức cho phiên bản v1.

## 1. Bản đồ cấu trúc (Sitemap)
- **Getting Started**: Hướng dẫn cài đặt CLI và Webview.
- **Platform Overview**: Kiến trúc tổng quan đa tác nhân.
- **Runtime Architecture**:
  - Resident Runtime Manager (Watchdog)
  - Resident Orchestrator (Daemon)
  - Ephemeral Subagents (Workers)
- **Runtime API Reference**: Đặc tả chi tiết các hợp đồng giao tiếp (WebSocket, Named Pipes).
- **Runtime SDK Reference**: Thư viện tích hợp dành cho các nhà phát triển Skill.
