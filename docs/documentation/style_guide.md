# Documentation Style Guide — Runtime Platform v1

Quy chuẩn trình bày tài liệu kỹ thuật dành cho nhà phát triển nền tảng.

## 1. Thuật ngữ chuẩn hóa
- **Resident Orchestrator**: Dịch vụ nền trú đóng ngầm dài hạn (Daemon).
- **Resident Runtime Manager**: Tiến trình giám sát tự phục hồi (Watchdog/Supervisor).
- **Subagent**: Tác nhân con thực thi tác vụ (Worker).

## 2. Cách thức định dạng và sơ đồ
- Bắt buộc vẽ sơ đồ Mermaid mô tả luồng dữ liệu khi đề xuất thay đổi API.
- Định dạng code block Python và JSON tuân thủ chuẩn GFM.
