# Reusable Component Inventory — Interactive Docs

Danh mục các cấu phần tương tác dùng chung cho trang tài liệu.

## 1. Cấu phần hiển thị và tương tác
- **CodeSnippetContainer**: Hiển thị code, hỗ trợ Copy-to-clipboard, Tab ngôn ngữ (Python, JS, CLI).
- **DAGExplorer**: Render đồ thị DAG của các Subagents bằng SVG tương tác, cho phép click xem dependency và scope lock.
- **StateViewer**: Cấu phần JSON tree view trực quan hiển thị live state của `agents.json` và `tasks.json`.
