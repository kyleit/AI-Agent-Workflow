# Interactive Docs Redesign — Runtime Platform v1

Đặc tả thiết kế lại trải nghiệm tương tác học tập của hệ thống tài liệu.

## 1. Mô hình tương tác chính
- **Bộ khám phá vòng đời (Lifecycle Explorer)**: Người dùng có thể click vào từng giai đoạn khởi tạo, lập kế hoạch, phát hành để xem hoạt động tương ứng của Resident Orchestrator.
- **API Explorer**: Giao diện thử nghiệm gửi lệnh JSON-RPC ảo trực tiếp trên Webview và xem trạng thái phản hồi tức thì.
