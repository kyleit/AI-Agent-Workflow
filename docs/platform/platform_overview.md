# AIWF Platform Ecosystem v1 Overview

Hệ sinh thái nền tảng AIWF được xây dựng trên hạ tầng Runtime API v1 ổn định.

## 1. Thành phần hệ sinh thái
- **Trình điều phối trú đóng (Resident Orchestrator)**: Quản lý DAG và phân bổ worker.
- **Trình giám sát (Runtime Manager)**: Watchdog tự phục hồi và Resource Governor.
- **SDK & Marketplace**: Cho phép mở rộng và phát hành Skills an toàn.

## 2. Mục tiêu kiến trúc
- Cung cấp trải nghiệm phát triển nhất quán.
- Đảm bảo an toàn phân quyền và cô lập tài nguyên.
