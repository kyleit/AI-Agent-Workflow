# AIWF Skill Certification Process

Quy trình thẩm định và cấp chứng nhận tương thích cho các Skills.

## 1. Tiêu chí đánh giá chất lượng
- **API Compliance**: Không gọi trực tiếp tệp tin hệ thống trái phép, bắt buộc đi qua SDK.
- **Resource Limits**: Tiêu hao bộ nhớ và CPU nằm trong mức an toàn cho phép.
- **Verification Tests**: Đầy đủ unit tests đạt độ phủ > 80%.

## 2. Phân loại xếp hạng
- **Certified**: Đạt mọi tiêu chí bảo mật, hiệu năng và tương thích API.
- **Compatible**: Tương thích API cơ bản nhưng cần tối ưu hiệu năng.
- **Deprecated**: Sử dụng các phương thức cũ lỗi thời và sẽ bị xoá bỏ.
