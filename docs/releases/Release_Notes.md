# AIWF OS v1.0.0 Release Notes

## 1. Summary
Bản phát hành chính thức đầu tiên của hệ điều hành AIWF OS v1.0.0, thiết lập nhân máy trạng thái Executive Runtime, cô lập workspace VFS, websockets daemon và Agent SDK.

## 2. New Features
- **Executive Loop (FEAT-086)**: Nhân máy trạng thái xử lý objective.
- **Task Graph (FEAT-087)**: Scheduler phân bổ task theo thứ tự ưu tiên DAG.
- **VFS Overlay (FEAT-098)**: RAM-backed ảo hóa thao tác I/O.
- **WebSocket IPC Sidecar (FEAT-099)**: Daemon stream log thời gian thực.
- **Agent Registry (FEAT-088)**: SDK tích hợp Agent của bên thứ ba.
